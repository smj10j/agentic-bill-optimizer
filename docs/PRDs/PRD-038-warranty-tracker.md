# PRD-038 — Warranty & Return Tracker

**Priority**: P3
**Status**: Draft
**Registry**: [REGISTRY.md](./REGISTRY.md)

---

## Problem Statement

Users forget about return windows, warranty periods, and price protection deadlines — leaving money on the table. The agent should track these time-sensitive opportunities and remind users before they expire.

---

## Key Ideas

- Purchase date tracking linked to transactions
- Return window countdown with timely reminders
- Warranty period alerts before expiration
- Price protection monitoring for eligible purchases
- Extended warranty evaluation (worth it or not)
- Receipt-linked tracking for proof of purchase
- Proactive "return by X" reminders with action links
- Claim assistance with guided steps for warranty and returns

---

## Success Metrics

- Tracked items per user (purchases with active windows)
- Reminder-to-action rate (% of reminders that lead to returns/claims)
- Dollars recovered via returns, warranty claims, and price protection
- Coverage rate (% of eligible purchases tracked)
- User-reported satisfaction with reminder timing
