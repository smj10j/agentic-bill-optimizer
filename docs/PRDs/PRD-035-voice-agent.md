# PRD-035 — Voice Agent Interface

**Priority**: P3
**Status**: Draft
**Registry**: [REGISTRY.md](./REGISTRY.md)

---

## Problem Statement

Sometimes typing isn't convenient — while driving, cooking, or just being lazy. A voice interface to the financial agent makes it accessible in more contexts. "Hey Orbit, how much did I spend on food this week?" should just work.

---

## Key Ideas

- Speech-to-text integration for voice input
- Voice command grammar for common financial queries
- Natural language understanding tuned for financial intent
- Voice response via text-to-speech for hands-free interaction
- Hands-free bill approval with voice confirmation
- Voice authentication for secure actions
- Smart speaker integration (Alexa, Google Home, Siri)
- Accessibility benefits for visually impaired users

---

## Success Metrics

- Voice query success rate (% understood and answered correctly)
- Voice session frequency per user
- Task completion rate via voice vs. text
- Voice-originated actions (bill pay, approvals) per month
- Accessibility user adoption and satisfaction
