# PRD-034 — Subscription Alternative Finder

**Priority**: P2
**Status**: Draft
**Registry**: [REGISTRY.md](./REGISTRY.md)

---

## Problem Statement

Users often stick with the first service they subscribed to even when better or cheaper alternatives exist. The agent should proactively suggest alternatives when it detects a subscription that could be replaced with a better-value option.

---

## Key Ideas

- Service category mapping (streaming, productivity, fitness, etc.)
- Feature comparison between current and alternative services
- Cost comparison with annualized savings projections
- Migration assistance with step-by-step switch guides
- Trial coordination to test alternatives risk-free
- "Switch and save" calculations with confidence intervals
- Satisfaction tracking post-switch to validate recommendations
- Bundle optimization across multiple subscriptions

---

## Success Metrics

- Alternative suggestions surfaced per user per month
- Switch rate (% of suggestions acted on)
- Average monthly savings per completed switch
- Post-switch satisfaction score (user-reported)
- Subscription cost reduction across the user base (aggregate)
